## 3 在``gtl_config.json``中设置窗口参数:

```json
{
        "宽度": 800,
        "高度": 600,
        "最低透明度": 0.6,
        "渐变": true,
        "源语言": "en",
        "目标语言": "zh-CN"
}
```
## 4 打包:
前提：已经安装了pyinstaller

```
python unpack2exe.py

```
